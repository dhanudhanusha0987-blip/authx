Access Control

Access Control defines who can access resources and what actions they are allowed to perform within a system.

👤

Role-Based Access Control (RBAC)

Access is granted based on predefined user roles.

 🏷️

Attribute-Based Access Control (ABAC)

Access decisions are made using attributes.

 🔒

Principle of Least Privilege

Users receive only the minimum access required.

 📊

Access Monitoring & Auditing

Tracks and reviews access activity.
